# Инструкции разработки проекта NEXTWEB

## 1. Генерация ассетов с помощью GPT Image 2
* ИИ обязан использовать скилл /gpt-image для генерации графики, когда пользователь просит добавить картинки, иконки или иллюстрации.
* Все API-ключи и эндпоинты берутся исключительно из файла .env в корне проекта (переменные OPENAI_API_KEY и OPENAI_API_BASE).
* Сгенерированные картинки сохранять строго в папку public/img/ или public/images/.
* При повторных генерациях сохранять единый визуальный стиль, используя флаг -i с путями к уже утвержденным картинкам-референсам.

## 2. Создание видео и анимаций через Remotion
* Для создания сложных зацикленных видео-фонов, анимаций открытия и промо-роликов ИИ может использовать библиотеку emotion (уже добавлена в devDependencies).
* Шаблон кода анимации на React должен размещаться в src/remotion/.
* Чтобы отрендерить видео-файл, ИИ должен использовать команду:
  
px remotion render src/remotion/[название_файла].tsx public/video/[название_видео].webm
* Рекомендуемый формат для сайта — .webm (он весит намного меньше, чем .mp4, и поддерживает прозрачный фон).

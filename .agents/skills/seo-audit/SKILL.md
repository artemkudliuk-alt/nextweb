# SEO Audit Skill

Build indexable, search-engine-friendly web pages.

## Rules & Principles

1. **Semantic Hierarchy**:
   - Every page must have exactly one `<h1>` containing the main keyword or primary service title.
   - Use `<h2>` for major sections and `<h3>` for nested content blocks.

2. **Metadata & Headings**:
   - Ensure document titles and meta descriptions are set dynamically based on page content.
   - Add descriptive alt attributes to all image elements.

3. **Schema Markup**:
   - Implement Schema.org microdata (e.g. JSON-LD for Services or Products) to help search engines understand the structure of the data.

4. **Performance for SEO**:
   - Optimize images (size, lazy loading below the fold).
   - Ensure fast Core Web Vitals (minimize layout shifts via defined width/height parameters, fast page loading).

# Creative Ideation & Design Concepts

Always brainstorm creative concepts and outline interactive options before writing code for any page. Provide the user with visual layout ideas, metaphors, and third-party tool suggestions.

## Rules & Principles

1. **The Three-Concept Framework**:
   For any new page or major section redesign, brainstorm 3 distinct design concepts for the user:
   - **Concept A: Minimalist Tech Blueprint** (thin lines, grid lines, monospace annotations, layout blueprint feel).
   - **Concept B: Cinematic Storytelling** (spacious layouts, large immersive typography, bold oklch accents, scroll-triggered text split reveals).
   - **Concept C: High-Fidelity Interactive Dashboard** (glassmorphic control panels, dynamic calculation widgets, active states, mock charts or counters).

2. **WebGL & 3D Options (Spline / Three.js / Shaders)**:
   - Suggest 3D objects or layouts (e.g., interactive abstract spheres, rotating nodes, floating cyber rings).
   - Suggest Spline for fast, interactive WebGL integrations, or custom canvas elements with GLSL fragment shaders (e.g., fluid grid particles) for background depth.

3. **Interactive Vector Motion (Rive / Lottie)**:
   - Suggest Rive for complex interactions that respond to mouse coordinates (e.g. an interactive robot face, button hover loops, or animated illustrations).
   - Suggest Lottie for lightweight micro-animations (e.g. checkmark confirmations, loading loops, success states).

4. **Advanced Layout Patterns**:
   - Suggest layout architectures like:
     - **Sticky Card Deck**: Cards stack on top of each other as the user scrolls down.
     - **Horizontal Timeline**: Scroll changes vertical scroll into horizontal slide for chronologies.
     - **Split-Screen Panel**: Pinned media column on one side, scrolling text on the other.

5. **Third-Party Visual Tools**:
   - Suggest external resources if native CSS is limited:
     - CSS Shader generators (ShaderToy / Canvas shaders).
     - Spline UI builder for 3D layouts.
     - Font pairings (Cabinet Grotesk + Satoshi / Geist Mono).
     - SVG path builders for custom drawing layouts.

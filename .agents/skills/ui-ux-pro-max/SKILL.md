# UI / UX Pro Max Skill

Deliver elite user experiences with clean layout patterns and consistent styles.

## Rules & Principles

1. **Color Systems**:
   - Limit the color scheme to a neutral dark/light base with one primary accent.
   - Use OKLCH variables for precise color transitions and proper contrast ratios.

2. **Component States**:
   - Ensure every interactive element implements its 8 states: default, hover, active, focus, disabled, loading, error, success.
   - Give buttons clean visual active (press) states and outline focus rings.

3. **Responsive Grids**:
   - Use grid-based spacing (e.g. multiples of 4px/8px).
   - Ensure all grids collapse smoothly on mobile viewports.
   - Use min-h-[100dvh] on full-screen segments to prevent layout shifts from dynamic browser toolbars.
